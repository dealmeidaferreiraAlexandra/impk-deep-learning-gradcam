# 🔥 Grad-CAM Visualizer

An interactive deep learning project that explains **how neural networks see images** using Grad-CAM.

---

## 🌐 Live Demo

👉 [Try the app](https://impk-deep-learning-gradcam-fncpm6mfbm7ycb6b7dsdsp.streamlit.app)

---

## 🧠 What this project does

* Classifies images using a CNN trained on CIFAR-10
* Visualizes model attention using Grad-CAM
* Shows top predictions with confidence
* Allows real-time interaction via Streamlit

---

## 🎯 Why this matters

Deep learning models are often black boxes.
This project makes them **interpretable**, showing *where* and *why* the model is making decisions.

---

## 🚀 Features

* 🔥 Grad-CAM heatmaps
* 📊 Top-3 predictions
* 🎚️ Adjustable attention intensity
* 💾 Download results
* 🌐 Interactive web app

---

## 🛠 Tech Stack

* Python
* PyTorch
* Streamlit
* OpenCV
* NumPy

---

## ▶️ Run locally

```bash
git clone https://github.com/dealmeidaferreiraAlexandra/impk-deep-learning-gradcam.git
cd impk-deep-learning-gradcam

python -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt
streamlit run app.py
```

## ⚠️ Notes

* Model trained on CIFAR-10
* Predictions may vary on real-world images
